# Subject Line Scorer

## What it is
A single-file HTML tool for scoring and generating cold email subject lines.
Built for Sachin's sales workflow at Freshworks.

## How to run
Open `index.html` directly in any browser — no build step, no dependencies.

## Key files
- `index.html` — the entire app (HTML/CSS/JS, self-contained)

## Features
- **Generate tab** — input prospect context, get 6 ranked subject lines
- **Score tab** — paste any subject line, get scores on 5 dimensions + alternatives
- **Patterns tab** — cheat sheet of formulas that work

## Scoring dimensions (each /10, total /50)
1. Personalisation
2. Curiosity
3. Clarity
4. Length (≤8 words = 10/10)
5. Spam Risk
